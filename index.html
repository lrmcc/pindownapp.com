About

PinDown is a Boulder tech startup. Our app/web service is an event service designed for CU Boulder community. We collect all of the public campus events and put it on our app, a singular platform and a shared calendar. We have the ambition to connect both event holders/organizations and event viewers/individuals, or event viewers with other event viewers through attending events. We believe PinDown serves the mission of maximally benefiting people's career pursuit and social life in events they choose to participate. Thus, we call this app "A Campus Event Platform to Improve the Experience of Real Life Engagements."

Functionalities and Features

The map based platform

PinDown is a map based event platform with Google maps. We want to provide the experience of visualization when checking out pins/events.

One pin, one event

Each pin shown on the map indicates an event. When clicking a pin, an info window that contains event details will pop up.

User's location

Users can choose to show their own location on the map. They can know the estimated distance of events around them.

Pins filter system

Every event belongs to a category. Users can distinguish the pins and find out events quicker by selecting different categories of events.

Number of views

We collect the number of views when every time the event info window is opened. The count won't increase when same user open the window repeatedly.

The state of events

Events that have happened: Pins will always disappear when the events are expired.
Events that are happening: The color of pins will be green.
Event that will happen later: The color of pins will be red.
